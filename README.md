# marburg-virus-awareness
A web-based platform designed to educate the public, spread accurate information, and promote preventive measures against the Marburg Virus Disease (MVD).
This project aims to raise awareness, reduce misinformation, and help communities stay safe through verified health information and easy-to-understand content.

# Features

Home Page – Clear introduction to Marburg Virus

Symptoms – Visual and descriptive list of early & late symptoms

Prevention – Protective measures and safety guidelines

Transmission – How the virus spreads

Treatment Information – Supportive care and current medical recommendations

Emergency Contacts – Quick access to help centers

Responsive UI – Works on all devices

Easy Navigation – Simple and user-friendly design

# Purpose of the Project

The main objective of this website is to:

Provide accurate, reliable, and verified information about Marburg Virus

Raise awareness in communities that may be at risk

Help prevent the spread of misinformation

Educate the public using a simple, mobile-friendly web interface
